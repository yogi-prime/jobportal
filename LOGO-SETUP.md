# Career Hub - Logo Setup Guide

## 🎨 Automatic Logo Installation

When you activate the **Career Hub** theme, the logo will **automatically** be uploaded to your WordPress Media Library and set as your site logo!

## 📁 Logo Files Included

The theme includes two logo files in the `assets/images/` folder:

1. **`logo.png`** - Horizontal logo for header (300x100px recommended)
2. **`logo-vertical.png`** - Vertical logo for footer (200x200px recommended)

## ✅ What Happens on Theme Activation?

When you activate the theme for the first time:

1. ✅ Logo automatically uploads to **Media Library**
2. ✅ Logo automatically sets as **Site Logo** in Customizer
3. ✅ Logo displays in **Header** and **Footer**

## 🔧 Manual Logo Change (Optional)

If you want to change the logo later:

1. Go to **Appearance → Customize**
2. Click **Site Identity → Logo**
3. Upload your custom logo
4. Save changes

## 📝 Logo Specifications

### Header Logo (Horizontal)
- **Format:** PNG, SVG, or JPG
- **Recommended Size:** 300x100px
- **Max Height:** 50px (auto-scaled)
- **Background:** Transparent PNG works best

### Footer Logo (Vertical)
- **Format:** PNG, SVG, or JPG
- **Recommended Size:** 200x200px or 140x160px
- **Max Height:** 140px
- **Color:** White/light colors (footer has dark background)

## 🎯 Pro Tips

- Use **PNG with transparent background** for best results
- Keep logo **simple and readable** at small sizes
- Test on **mobile devices** to ensure visibility
- Logo should be **web-optimized** (under 200KB)

## 🆘 Troubleshooting

**Logo not showing?**
1. Clear browser cache (Ctrl + Shift + Del)
2. Go to **Appearance → Customize → Site Identity**
3. Re-upload logo manually
4. Hard refresh page (Ctrl + F5)

**Logo too big/small?**
1. Edit logo size in `header.php` (change `height="50"`)
2. Or use CSS: `.jobportal-logo img { height: 60px; }`
