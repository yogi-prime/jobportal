VERSIONING SYSTEM
-----------------
We use Semantic Versioning (SemVer):
   - MAJOR version (2.0.0) - Incompatible changes or major new features
   - MINOR version (2.1.0) - New features, backward compatible
   - PATCH version (2.0.1) - Bug fixes, backward compatible
