JobPortal - Premium SaaS & Startup Landing Page WordPress Theme

== Description ==

JobPortal is a premium, high-converting SaaS & Startup Landing Page WordPress Theme built for AI tools, software products, mobile apps, digital agencies, and tech startups.

= Key Features =

* One-click demo import
* 50+ Customizer options
* WooCommerce ready
* SEO optimized
* Schema markup
* RTL support
* Blazing fast performance
* Mobile responsive
* Custom post types (Services, Testimonials, Pricing, FAQs)
* AJAX contact form
* Newsletter subscription
* Mega menu support
* Multiple header layouts
* Multiple footer layouts
* Preloader with 4 styles
* Smooth animations (AOS + GSAP)

== Installation ==

1. Upload the theme folder to /wp-content/themes/ directory
2. Activate the theme through 'Appearance > Themes' menu in WordPress
3. Follow the setup wizard to install recommended plugins
4. Import demo content (optional)
5. Customize via Appearance > Customize

== Customization ==

= Change Logo =
1. Go to Appearance > Customize
2. Click on Site Identity
3. Upload your logo image
4. Save changes

= Edit Text Content =
1. Go to Appearance > Customize
2. Navigate to sections (Hero, Features, Pricing, etc.)
3. Edit text fields
4. Click Publish

= Change Colors =
1. Go to Appearance > Customize
2. Click on Theme Colors
3. Use color pickers to change colors
4. Click Publish

= Add Services/Testimonials =
1. Go to Services or Testimonials menu in WordPress admin
2. Click Add New
3. Fill in the details
4. Click Publish

== Frequently Asked Questions ==

= How do I import demo content? =

Install the "One Click Demo Import" plugin, then go to Appearance > Import Demo Data and click Import.

= Is this theme compatible with page builders? =

Yes, JobPortal works with Elementor and Gutenberg block editor.

= Does it work with WooCommerce? =

Yes, the theme is fully WooCommerce compatible.

= Can I translate this theme? =

Yes, JobPortal is translation-ready and includes a .pot file.
