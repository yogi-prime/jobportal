## Installation

### Method 1: WordPress Dashboard
1. Go to **Appearance → Themes → Add New**
2. Click **Upload Theme**
3. Choose `jobportal-elite.zip`
4. Click **Install Now**
5. Click **Activate**

### Method 2: FTP Upload
1. Unzip `jobportal-elite.zip`
2. Upload `jobportal` folder to `/wp-content/themes/`
3. Go to **Appearance → Themes**
4. Activate **JobPortal Elite**

---

## Quick Start Guide

### Step 1: Install Recommended Plugins
After activation, you'll see a notice to install recommended plugins:
- One Click Demo Import
- Elementor
- Contact Form 7
- WooCommerce (optional)

### Step 2: Import Demo Content
1. Go to **JobPortal Elite → Demo Import**
2. Choose a demo (Corporate, Startup, or Freelance)
3. Click **Import**
4. Wait 2-3 minutes
5. Done! Your site looks like the demo

### Step 3: Customize Your Site
1. Go to **Appearance → Customize**
2. Customize:
   - Logo & site identity
   - Colors & typography
   - Header & navigation
   - Footer settings
   - Homepage sections

### Step 4: Add Unique Features to Pages
1. Create pages for each feature:
   - "Build Your Resume" → Add `[jobportal_resume_builder]`
   - "Find Your Match" → Add `[jobportal_job_matcher]`
   - "Employer Dashboard" → Add `[jobportal_ats_dashboard]`
   - "Salary Calculator" → Add `[jobportal_salary_calculator]`
   - "Schedule Interview" → Add `[jobportal_interview_scheduler]`

2. Add these pages to your menu

3. Done! Users can now access all elite features.
